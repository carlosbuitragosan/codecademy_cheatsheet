CSS Cheatsheet

Property Description Example Explanation
@font-face Defines custome fonts for web usage @font-face {
font-family: 'MyCustomFont';
src: url('fonts/MyCustomFont-Regular.woff2') format('woff2'),
url('fonts/MyCustomFont-Regular.woff') format('woff');
} 		@font-face Block: Defines a custom font named MyCustomFont.
font-family: The name you will use to reference the font in your CSS.
src: The source of the font file(s). Here, it's specified in both woff2 and woff formats for better browser compatibility.
border-collapse Creates a single border in tables table {
border-collapse: collapse;
}		 It applies to tables. Values are 'collapse' or 'separate' (default).
background Shorthand for setting myltiple background-related properties for an element. background: url(image.png) center/cover no-repeat green Syntax:
background: [background-color] [background-image] [background-position] / [background-size] [background-repeat] [background-origin] [background-clip] [background-attachment];

Notice that bg-position & bg-size are set together with '/'

Note: the browser is able to recognise different syntax order
::before Inserts content 'before' the content of an element. It is tipically used to add cosmetic content to an element. .image-container {
background: url(image.omg)
position: relative;
}
.image-container::before {
content: '';
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
background-color: rgba(0, 0, 0, 0.5);
}		 .image-container: Positioned relative to establish a containing block for absolutely positioned elements.
.image-container::before: Creates a pseudo-element (::before) that spans the entire dimensions of .image-container.
content: '': Required for the pseudo-element to generate content.
position: absolute;: Positions the pseudo-element absolutely relative to .image-container.
top: 0; left: 0; width: 100%; height: 100%;: Covers the entire area of .image-container.
background-color: rgba(0, 0, 0, 0.5);: Sets a semi-transparent black overlay using RGBA values (rgba(red, green, blue, alpha)), where alpha controls opacity (0.5 means 50% opaque).
